## pratica de estudos javascript-linguagem

### anotações
- javascript puro e node.js; 
- REPL no NodeJS significa Read Eval Print Loop (representa um console de computador);
- camelCase, operações aritméticas; 
- JavaScript é uma linguagem fracamente tipada, case senstive; 
- declarar texto entre crases, concatenações, interpolação
- constant é um tipo de variável que não pode ser modificada uma vez que sua atribuição original foi feita;
- let é um tipo de variável que varia; 
- gerar listas a partir do array, adciona item push, remove item splice; 
- while: operador de comparação (< [menor], > [maior], <= [menor ou igual], >= [maior ou igual], == [igual a] e != [diferente de]) e qualquer operador lógico (&& [and], || [ou]);
- breack, debbuger, while, for loop;
